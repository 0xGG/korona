# Korona

**Korona = Kill Corona (virus)**, to memorate those who fight against the corona virus bravely in 2020.  
仅已此库纪念 2020 年那些勇敢与新冠状病毒抗争的人们。

Minimal library to quickly construct decentralized p2p network
